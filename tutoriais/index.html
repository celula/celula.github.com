Instalação de dependências

O Célula se apoia em diversos softwares open source e, dessa maneira, é necessário que eles estejam presentes para a utilização. As bibliotecas menores estão incluídas no download.

Softwares que não estão incluídos no download e que precisam ser instalados:

  • MySQL
  • Node.js
  • Zend Framework
